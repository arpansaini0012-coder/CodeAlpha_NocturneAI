# 🎼 Nocturne AI - Deep Learning Sequential Music Generation Interface

An advanced neural network orchestration simulator designed to pre-process musical data structures and train sequential generation loops. This dashboard tracks deep learning matrix steps for **Task 3: Music Generation with AI** under the **CodeAlpha Internship Program** evaluations.

---

## 🚀 Technical Architecture Matrix (Ref: image_18373b.png)
* **MIDI Processing Analytics (Task Requirement 1 & 2)**: Structural token mapping emulates `music21` python conversions, creating safe scalable node data packages.
* **Deep Learning Structural Switch (Task Requirement 3 & 4)**: Interface supports hyperparameter configurations for standard **LSTM Recurrent Neural Networks (RNNs)** and **Generative Adversarial Networks (GANs)** layers.
* **Dynamic Synthesis Web Playback (Task Requirement 5)**: High-precision rendering engineered directly via native Web Audio API oscillators, enabling instantly simulated track outputs and dynamic vector data downloads.
* **Adaptive Theme Matrix UI**: Features a sleek, industry-standard modern dark interface configuration with responsive custom global client toggles (Light/Dark themes).

---

## 🛠️ Hyperparameter Settings Explained
* **Temperature**: Adjusts pattern mathematical mutations ($0.20 \text{ to } 1.50$) to handle output diversity.
* **Sequence Length**: Restructures the depth array bounds ($16 \text{ to } 64$ parallel structural note steps).
* **BPM Controls**: Alters sample streaming rates continuously without freezing system loops.
